## What it is
**Build with Claude Code** — a 4-week, live, cohort-based program. You walk in
with an idea and walk out with **three real AI products on live URLs** (including
your own idea). You don't write code — you direct Claude Code, which writes it.

## Who it's for
AI-curious non-coders (designers, marketers, ops, founders, generalists, students),
ChatGPT/Claude power users who want to ship products others use, and PMs/operators/
analysts who want to ship without waiting on engineering. People who took the free
course and want structured follow-through.

## Who it's NOT for
Passive video-only learners; people who want someone to build their idea for them;
senior software engineers wanting a deep technical AI/ML course; anyone who can't
commit 2–4 hours/week of build time.

## What you build (3 products)
1. **Second Brain** (Week 1) — single-user personal knowledge tool; tag, summarise, answer with citations.
2. **AI Interview Coach** (Week 2) — multi-user B2C app; three Claudes (interviewer, grader, coach); the headline build.
3. **Your own idea** (Weeks 3–4) — scoped to a 7-day v1, deployed in the right shape, used by a stranger before Demo Day.

Core frameworks taught: the 5-step build loop (scaffold → implement → review →
refactor → ship); the CLAUDE.md system; the agent-architecture decision tree
(single call / pipeline / sub-agents / Multi-Agent Orchestration); the product-
shape menu (web app / background agent / MCP server / Skill / Slack bot / CLI).
Also: MCP, Skills 2.0, hooks, auth (Clerk) + data isolation (Supabase RLS),
evals (LLM-as-judge), cost engineering (prompt caching, batch API, model routing).

## The stack
Claude Code (build agent), Next.js + Vercel (web + deploy), Supabase (Postgres),
Anthropic SDK (AI runtime), Clerk (auth), MCP (connectors), Skills 2.0,
Anthropic Multi-Agent Orchestration, Claude Agent SDK (non-web shapes).
Pre-cohort setup ~1 hour. Requires: laptop (Mac/Linux preferred, Windows via WSL),
Claude Pro ($20/mo) + Claude Code, and free tiers of GitHub/Vercel/Supabase/Clerk.

## Time commitment
4–6 hours/week: ~1.5h live session, 30–60 min pre-reads, 2–4h building. Weeks 2–3
are most intensive (some put in 8–10h). ~15–25 hours total over 4 weeks.
